### About the Experiment 

This experiment helps learners understand and compare the growth in algorithm running time as input size increases. The simulation demonstrates the practical behavior of common complexity classes, including logarithmic, linear, linearithmic, quadratic, cubic, exponential, and factorial, using algorithms such as Binary Search, Linear Search, Quick Sort, Bubble Sort, Matrix Addition, Matrix Multiplication, Subset Generation, and Permutation Generation.
